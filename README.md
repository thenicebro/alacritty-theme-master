```bash
	Usage: alacritty_themes [OPTION]...
	
	OPTION: 
		[EMPTY]: randomly switch into the themes

		all/-a/--all: show all themes name

		help/-h/--help: show help information

		show/--show: show main information of config file

		size/-s/--size [FONT_SIZE]: change the alacritty font size

		theme/-t/--theme [THEME_NAME]: switch into the special themes
```
